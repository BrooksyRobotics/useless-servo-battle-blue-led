# useless-servo-battle-blue-led
Two servos fight over the LED being on and off until they "blow" it up.  Code controls servos and LED.

Buttons are just props.  All the LED controls are driven by code and timing with servos.

## Hardware:
  
Positional servos needed to receive angle controls
 - Servo A   → D9   - presses the OFF button (right servo)
 - Servo B   → D10  - presses the ON  button (left servo)
 - LED       → D6   - **100 ohm resistor
